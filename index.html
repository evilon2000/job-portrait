.NET 侧定位

偏个人完整交付，擅长中后台业务系统、通用能力沉淀、模块复用与持续迭代，编码习惯较稳定，具备较好的自驱交付能力。

Java 侧定位

偏团队协作与平台建设，覆盖农业 SaaS、GIS、园区/社区治理、流程与数据统计类系统，能在复杂业务域中推进接口、服务、数据模型与模块拆分。

求职表达重点

不把自己包装成纯算法或纯框架工程师，而是强调“能落业务、能带结构、能识别风险、能长期维护”的工程型开发者形象。
