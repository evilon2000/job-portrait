# Personal Developer Portrait

用于求职展示的静态仓库，内容来自本地开发 wiki 的结构化整理。

## 在线地址

- 主页: `https://evilon2000.github.io/job-portrait/`
- English: `https://evilon2000.github.io/job-portrait/index-en.html`
- `.NET` 画像: `https://evilon2000.github.io/job-portrait/personal-dev-portrait.html`
- `Java` 画像: `https://evilon2000.github.io/job-portrait/java-team-dev-portrait.html`

## 中文说明

这个仓库把能力拆成两条独立表达线:

- `.NET`: 强调个人开发、业务落地、稳定交付、较好的编码习惯
- `Java`: 强调团队协作、平台后端、复杂业务建模、多模块项目推进

这样的拆分可以避免不同技术栈互相污染，更接近真实履历表达。

### 适合的使用方式

- 对外统一发主页
- 投 `.NET` 岗位时补发 `.NET` 画像页
- 投 `Java` 岗位时补发 `Java` 画像页

## English Summary

This repository is a static career landing page built from a local code wiki.

It separates capability presentation into two tracks:

- `.NET`: personal delivery, business implementation, stable coding habits
- `Java`: team collaboration, platform backend work, complex business systems

### Recommended Usage

- Share the homepage as the main entry
- Send the `.NET` page for .NET backend roles
- Send the `Java` page for Java backend roles
