# Personal Developer Portrait

这是一个用于求职展示的静态仓库，内容来自本地开发 wiki 的整理结果。

## 页面入口

- 首页：`index.html`
- `.NET` 个人画像：`personal-dev-portrait.html`
- `Java` 团队画像：`java-team-dev-portrait.html`

## 技术定位

### .NET

- 偏个人开发与完整交付
- 关注业务抽象、模块复用、稳定交付与良好编码习惯
- 适合中后台、SaaS、管理系统类岗位表达

### Java

- 偏团队协作与平台型后端建设
- 覆盖农业 SaaS、GIS、政企/园区/社区治理、统计与流程类系统
- 强调复杂业务建模、模块推进、系统整合与交付能力

## 使用方式

- GitHub Pages 根地址作为主页入口
- 投递不同岗位时，按目标栈补发对应子页面链接
