# Style Code Showcase

This directory contains curated code samples that reflect personal coding style and technical judgment.

Rules:
- Keep `.NET` and `Java` fully separated.
- Only keep representative samples, not full project dumps.
- Pair code files with short notes explaining why each sample matters.

Structure:
- `dotnet/code/`: personal `.NET` style samples
- `dotnet/notes/`: matching notes for `.NET` samples
- `java/code/`: Java platform/team-style samples
- `java/notes/`: matching notes for Java samples

These samples are selected from the local personal code wiki and are intended for portfolio display.
