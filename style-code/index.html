让面试官先看到最有辨识度的代码

这里不展示整仓源码，而是只展示最能代表个人风格的精选样本。`.NET` 看结构化抽象、表达式拼装、规则与流程建模；`Java` 看平台型后端、区域隔离、GIS 集成、工作流查询与跨系统编排。目的不是证明“写过很多代码”，而是让对方快速判断“你是怎样组织复杂系统的”。
