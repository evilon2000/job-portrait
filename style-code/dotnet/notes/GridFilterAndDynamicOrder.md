# ZCYX Snippet: Grid Filter And Dynamic Order

## 为什么值得保留

这段代码把列表筛选和动态排序收口成扩展方法，直接体现了你在 `.NET Core` 阶段已经开始把查询能力本身做成基础设施。

## 它体现的风格

- 按筛选 DTO 逐步收缩条件
- 用表达式做动态排序
- 默认排序与自定义排序并存

## 今天仍然会保留什么

- `ApplyFilter` 这种查询入口
- `OrderByDynamic` 这种表达式驱动能力
- 把排序字段和排序方向作为外部输入

## 今天会改什么

- 会更注意字段白名单，避免任意属性字符串带来的脆弱性
- 可能会把更多过滤保持在数据库表达式层，而不是部分转到内存层
